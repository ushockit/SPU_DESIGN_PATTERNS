1. AbstractFactory - Есть 3 вида графических интерфейсов (Windows, MacOS, Linux). В каждом из них есть окна и кнопки. Ваша задача реализовать фабрику, которая позволит создавать окна, кнопки в зависимости от выбранного интерфейса
2. Adapter - У вас есть html содержимое <html><head><meta charset="utf-8"><title>Hello world</title></head><body></body></html>, ваша задача создать адаптер, который будет конвертировать html в json (нужна реализация)
3. Facade - Взять за основу задание по работе с API приватбанка. Создать сервис, который будет скрывать внутри себя логику, по работе с API.
